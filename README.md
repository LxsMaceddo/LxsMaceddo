## Development approach

When working on a project I focus on three things:

**Structure**  
Code should be organized, modular and readable.

**Performance**  
Systems should handle real player counts without unnecessary overhead.

**Longevity**  
Games evolve. Systems should be easy to expand without rewriting everything later.
